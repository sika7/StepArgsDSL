# 📚 StepArgs DSL 使用例集

## 🧠 1. 要約＋翻訳タスク（LLM連携）

— 要約 —
要約[文字数上限:200]
要約[対象文書:この文章は、AI技術の歴史と未来について…]

— 翻訳 —
翻訳[入力言語:日本語]
翻訳[出力言語:英語]
翻訳[内容:要約結果]

## 🌐 2. Web記事収集＋キーワード抽出

— 検索 —
検索[キーワード:生成AI]
検索[件数:5]

— 記事取得 —
記事取得[URL:https://example.com/article1]
記事取得[URL:https://example.com/article2]

— 抽出 —
抽出[対象:本文]
抽出[抽出タイプ:名詞頻出]

## ✍️ 3. ブログ生成タスク（マルチセクション構成）

— 構成作成 —
構成作成[タイトル:AIが変える教育現場]
構成作成[見出し数:3]

— 本文執筆 —
本文執筆[セクション1:AI導入の背景]
本文執筆[セクション2:現場の変化]
本文執筆[セクション3:未来の可能性]

— 校正 —
校正[文体:中学生でもわかる]
校正[トーン:やさしく]

## 💬 4. チャットボットの応答制御（引数スイッチ）

— 応答生成 —
応答生成[モード:感情的]
応答生成[言語:日本語]
応答生成[対象発言:なぜAIに仕事を奪われるの？]

— フィルター —
フィルター[禁止語:侮辱, 脅迫]
フィルター[長さ上限:100]

## 📊 5. データ可視化（数値→グラフ指示）

— データ取得 —
データ取得[ソース:analytics.csv]
データ取得[期間:2024-01〜2024-12]

— グラフ生成 —
グラフ生成[種類:折れ線]
グラフ生成[X軸:月]
グラフ生成[Y軸:PV数]

## 🤖 6. 自律エージェント処理例（分解と依存関係）

— 目標設定 —
目標設定[ゴール:新しいAIライブラリを調査し、選定する]

— タスク分解 —
タスク分解[対象:目標設定]
タスク分解[粒度:中]
タスク分解[出力形式:StepArgs DSL]

— 検索 —
検索[キーワード:軽量なAIライブラリ 2025]

— 評価 —
評価[基準:速度, メモリ, サポート状況]
評価[候補:LibraryA, LibraryB]

## 🧰 応用的な構造

- ステップ単位の並列処理・逐次処理
- `引数名:値` に式的記述（例：式評価、変数展開）
- DSLからJSON、YAML、関数呼び出しへ変換可能
